# 🚀 Sistema de Clientes Web

Sistema web profissional para gerenciamento de clientes, desenvolvido com Python Flask.

---

## 🌐 Acesse o sistema online
👉 https://SEU-LINK-AQUI.onrender.com

---

## 📌 Funcionalidades

- 🔐 Login de usuário
- 📊 Dashboard com total de clientes
- ➕ Cadastro de clientes
- 🔍 Busca de clientes
- ❌ Exclusão de clientes
- 💾 Persistência com banco de dados SQLite

---

## 🛠️ Tecnologias utilizadas

- Python
- Flask
- HTML5
- CSS3
- SQLite
- Gunicorn
- Render (deploy)
